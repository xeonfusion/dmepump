# DMEPUMP
DMEPUMP (C) 2025-26 John George K. algorithm for Target Control Infusion (TCI) calculation and simulation

Supports:
- Real-time Plotting
- Cp, Ce and infusion rate CSV data export
- Simulation logs export
- Effect site and Plasma site targeting
- Multi drug and multi model support
- Calculation of optimal loading bolus dose
- Calculation of optimal infusion rate to achieve target concentration
- Calculation of tpeak and ke0
- Multiple convergence methods
- Infusion rate limitation
- Custom simulation step time
- Fast-mode and real-time mode simulation
